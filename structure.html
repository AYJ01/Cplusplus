Basic Structure

Structure: C++ Language

The best way to learn a programming language is by writing programs. A common first program is "Hello World",
which prints "Hello World" on the screen.

Example Program:

// my first program in C++
#include <iostream>

int main() {
  std::cout << "Hello World!";
}

This simple program introduces fundamental C++ concepts:

Two slashes (//) indicate a comment. Comments do not affect program execution but help document the code.

The #include <iostream> directive imports standard input/output functionality, allowing the program to display output.

Blank lines improve readability and have no effect on program execution.

The main() function is where execution begins. Every C++ program must have a main function.

The opening brace { marks the start of the function, and the closing brace } marks its end.

std::cout << "Hello World!"; prints the message to the screen. The semicolon ; marks the end of the statement.

This breakdown introduces essential elements of C++ programming. Future lessons will cover more advanced topics.
